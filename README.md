# breakpad-cmake

**breakpad-cmake** is a lightweight CMake wrapper for [Google Breakpad](https://chromium.googlesource.com/breakpad/breakpad).

## Project Integration

### Usage with [FetchContent](https://cmake.org/cmake/help/latest/module/FetchContent.html)

```cmake
cmake_minimum_required(VERSION 3.14)
project(example)

include(FetchContent)
FetchContent_Declare(breakpad
    GIT_REPOSITORY "https://github.com/tayne3/breakpad-cmake"
    GIT_TAG "6662f7591ace2849a12a59b02ae639d95aef7393"
)
FetchContent_MakeAvailable(breakpad)

add_executable(example main.cpp)
target_link_libraries(example PRIVATE breakpad::breakpad)
```

### Usage with [CPM.cmake](https://github.com/cpm-cmake/CPM.cmake)

Use bundled breakpad source code:

```cmake
cmake_minimum_required(VERSION 3.14)
project(example)

include(cmake/CPM.cmake)
CPMAddPackage(
    NAME breakpad
    GIT_REPOSITORY "https://github.com/tayne3/breakpad-cmake"
    GIT_TAG "6662f7591ace2849a12a59b02ae639d95aef7393"
)

add_executable(example main.cpp)
target_link_libraries(example breakpad::breakpad)
```

Use specified version of breakpad source code:

```cmake
cmake_minimum_required(VERSION 3.14)
project(example)

include(cmake/CPM.cmake)
CPMAddPackage(
    NAME breakpad
    GIT_REPOSITORY "https://github.com/tayne3/breakpad-cmake"
    GIT_TAG "6662f7591ace2849a12a59b02ae639d95aef7393"
    OPTIONS 
    "BREAKPAD_USE_EXTERNAL ON"
    "BREAKPAD_EXTERNAL_URL https://chromium.googlesource.com/breakpad/breakpad"
    "BREAKPAD_EXTERNAL_TAG 2c736308b5a4c7a8371fa3a3e434f551eddd17c9"
)

add_executable(example main.cpp)
target_link_libraries(example breakpad::breakpad)
```

## Symbol & Minidump Tools

Analyzing the `.dmp` (Minidump) files generated by the client requires separate symbol extraction and processing utilities:

* **dump_syms**: Extracts Breakpad-format symbols from binaries containing debugging information.
* [mozilla/dump_syms](https://github.com/mozilla/dump_syms) (Cross-platform Rust implementation)

* **minidump_stackwalk**: Processes minidumps along with symbol files to produce human-readable stack traces.
* [rust-minidump](https://github.com/rust-minidump/rust-minidump) (Alternative stackwalking library and CLI)
